# RGB Ring Light  

### An ESP32C3 based RGB Ring Light

### Features  
* Clean Web based UI
* Realtime control
* Wireless (WiFi)
* Customizable Access Point credentials
* DarkMode
* Factory reset functionality
* OTA updates using compiled binary
